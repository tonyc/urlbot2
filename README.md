urlbot2
=======

Requirements 
------------

* Ruby 1.9.2 (use rvm! - https://rvm.beginrescueend.com/)
* Bundler (anything past 1.0.13 should be good, latest should be fine - http://gembundler.com)

Installation
------------

* bundle install

Running
-------

* $ NICK=urlbot SERVER=irc.example.com CHANNEL=#awesome ./urlbot.rb
